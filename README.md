# Home-credit-default-risk
ML project to predict loan default risk using random forest

# Home Credit Default Risk Prediction

## 📌 Problem
Predict whether a customer will default on a loan.

## ⚙️ Workflow
- Data Cleaning
- Feature Engineering
- SMOTE (handling imbalance)
- Model Training (Logistic Regression, Random Forest)
- Evaluation
- Feature Importance
- Cross Validation

## 📊 Results
- Random Forest Accuracy: 91%
- Logistic Regression Accuracy: 58%

## 📈 Key Insights
- Dataset is highly imbalanced
- Random Forest performs better
- External credit scores are most important features

## 🛠 Tech Stack
Python, Pandas, NumPy, Scikit-learn, Matplotlib
